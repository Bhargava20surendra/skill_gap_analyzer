# skill_gap_analyzer
Resume-based Skill Gap Analyzer using Python, Flask, MySQL, and NLP to recommend missing skills for job roles.
**Features**
User registration and login
Resume upload (PDF / DOCX)
Automatic skill extraction from resumes
Job role selection
Skill gap analysis
Missing skill recommendations
Analysis history tracking
Dark mode UI support

**Technologies Used**
__Backend__
Python
Flask
NLP (Skill extraction)
__Frontend__
HTML
CSS
JavaScript
__Database__
MySQL
__Libraries__
PyPDF2 / pdfplumber
docx2txt
Flask
MySQL Connector
**Project Architecture**
User → Upload Resume → Skill Extraction → Compare with Job Role Skills → Identify Missing Skills → Display Recommendations
